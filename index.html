Adhearsion

Adhearsion is...

Adhearsion is a new way to write voice-enabled applications. It's not just another API or library — it's a full-featured framework cleverly designed to both make your life easier and open up new possibilities. The name "Adhearsion" is a combination of "adhesion" and "hear" because Adhearsion shines best when adhering technologies together, especially with voice.

It's modern

Adhearsion uses the high-level Ruby programming language and boasts a comprehensive regression test suite. In the age of the social web, what other technologies are more social than voice? Adhearsion aims to help merge this so-called "Web 2.0" world with the previously untamable telephony world.

It's intuitive

Voice applications don't have to be hard. Take a look at some of the examples. The Ruby language is widely chosen for its inspiring code beauty and the Adhearsion framework takes this seriously. Cleaner or more beautiful code is easier to write and easier to maintain.

It's open-source

We take open-source software seriously. The telephony industry, above all, needs fresh and innovative talent and Adhearsion's here to bring it.

It's fun

Ever thought about controlling a phone call before? With Adhearsion, it's never been easier.
